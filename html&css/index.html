HyperText Markup Language

Estrutura básica do arquivo .html

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Minha página</title>
  </head>
  <body>
    <h1>Fique focado, e vai vencer no final.</h1>
  </body>
</html>

Categorias básicas

Elementos de bloco (block)

São elementos que, por padrão, iniciam em uma nova linha.

  • Parágrafos
  • Formulários
  • Tabelas

Elementos de linha (inline)

São elementos que, por padrão, não começam em uma nova linha, e ocupam somente a largura de seu conteúdo.

  • Imagens
  • Âncoras
  • Elementos de formatação

Trabalhando com Texto

Afinal, foi pra isso que o HTML foi criado.

As tags de texto

Os textos em HTML podem ser compostos por cabeçalhos (headers), parágrafos, quebras de linha, listas e tags de ênfase e importância.

Sou um cabeçalho nível 1

Sou um parágrafo. A seguir, uma lista não ordenada:

  • Bermuda
  • Sandália de couro

Sou um cabeçalho nível 4

  1. Cabeçalho
  2. Parágrafo, eu...
<h1>Sou um cabeçalho nível 1</h1>
<p>Sou um parágrafo. A seguir, uma lista não ordenada:</p>
<ul>
  <li>Bermuda</li>
  <li>Sandália de couro</li>
</ul>
<h4>Sou um cabeçalho nível 4</h4>
<ol>
  <li>Cabeçalho</li>
  <li>Parágrafo, eu...</li>
</ol>

As tags de ênfase e importância

São utilizadas quando queremos dar ênfase ou importância semântica a um trecho do texto.

Normal.

Com ênfase.

Com importância.

<p>Normal.</p>
<p><em>Com ênfase.</em></p>
<p><strong>Com importância.</strong></p>

Abalando as Estruturas

Ora, HTML não é só texto!

As tags de estrutura

  • header - representa um grupo de suporte introdutório ou navegacional.
  • nav - representa uma seção com links de navegação.
  • footer - representa um rodapé.
  • section - é um tematizador de conteúdo. Todo o conteúdo dentro de uma section deve tratar do mesmo assunto.
  • main - define o conteúdo principal dentro da body.
  • div - separador genérico em forma de bloco.

Tabelas

Representando dados em 2 dimensões

As tags para formar uma tabela

  • table - cria uma tabela
  • tr - adiciona uma linha à tabela
  • td - adiciona uma coluna à linha
  • thead, tbody, tfoot - separam as linhas de cabeçalho, corpo e rodapé, respectivamente, de uma tabela.
  • th - coluna que representa um cabeçalho.

Formulários

Queremos dados dos usuários!

As tags de formulário

  • form - cria um formulário
  • input - adiciona um campo de texto curto ao form
  • textarea - adiciona um campo de texto longo
  • select - adiciona um campo de seleção
  • label - adiciona um título ao campo
  • button - tag para botões

Cascade Style Sheets

Estrutura básica

seletor {
  propriedade: valor;
  propriedade: valor;

}
  • Seletor - tag html, class, id (Lista completa)
  • Propriedade - o que deseja modificar
  • Valor - modificação

Unidades

Tamanho

Em CSS utilizamos as propriedades width e height para definir largura e altura.

Seu documento é uma caixa!

Em CSS temos várias unidades de medida: px, pt, em, rem, %, entre outras.

  • padding: 10px 10px 10px 10px;
    • padding-top: 10px
    • padding-right: 10px
    • padding-bottom: 10px
    • padding-left: 10px
  • margin: 5px 10px
    • margin-top & margin-bottom: 5px
    • margin-right & margin-left: 10px
  • margin: 5px 7px 10px
    • margin-top: 5px
    • margin-right & margin-left: 7px
    • margin-bottom: 10px
  • border: 1px solid #333
    • border-width: 1px;
    • border-style: solid;
    • border-color: #333;
